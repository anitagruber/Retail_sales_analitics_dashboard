# Retail_sales_analitics_dashboard
A Streamlit-based retail sales analytics dashboard with KPI metrics, filtering options and Top 10 product analysis.
An interactive retail sales analytics dashboard built with Python and Streamlit.

## Overview

This project provides a business-oriented dashboard for analyzing retail sales performance.  
Users can filter data dynamically and explore key performance indicators and product performance insights.

## Features

- Interactive filters (Category, Sub-Category, Segment, Region)
- KPI metrics:
  - Total Sales
  - Total Profit
  - Distinct Orders
  - Selected Year
- Top 10 Products by Sales
- Top 10 Products by Profit
- Clean business-focused UI with custom styling

## Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib

1. Clone the repository
2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run app.py
```
